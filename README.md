## 🎁 How to Build and Run

### 🏡 Run Locally

```bash

part=part2 dotnet run
```

### 🐳 Run with Docker

```bash
docker build -t aoc24 .
docker run --rm -e part=part1 aoc24
docker run --rm -e part=part2 aoc24
```
